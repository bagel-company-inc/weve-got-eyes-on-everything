## Usage


Install [spatialite](https://www.gaia-gis.it/gaia-sins).


### Backend:

#### Setup:
```bash
cd backend
pip install -r requirements.txt
```

#### Create/Refresh Databases:

```bash
python3 refresh_databases.py
```

#### Start:
```bash
python3 app.py
```


### Frontend
#### Setup:
```bash
cd frontend
npm install
```

#### Start
```bash
npm start
```
